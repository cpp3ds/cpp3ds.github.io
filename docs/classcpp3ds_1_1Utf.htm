Detailed Description

template<unsigned int N>
class cpp3ds::Utf< N >

Utility class providing generic functions for UTF conversions.

cpp3ds::Utf is a low-level, generic interface for counting, iterating, encoding and decoding Unicode characters and strings. It is able to handle ANSI, wide, latin-1, UTF-8, UTF-16 and UTF-32 encodings.

cpp3ds::Utf<X> functions are all static, these classes are not meant to be instantiated. All the functions are template, so that you can use any character / string type for a given encoding.

It has 3 specializations:

  • cpp3ds::Utf<8> (typedef'd to cpp3ds::Utf8)
  • cpp3ds::Utf<16> (typedef'd to cpp3ds::Utf16)
  • cpp3ds::Utf<32> (typedef'd to cpp3ds::Utf32)
